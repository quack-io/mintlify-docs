## March 2026

<Steps>
  <Step title="Mar 9 — Messaging API v1.0">
    **Messaging API** `v1`

    - General availability of Tasks, Messages, and WebSocket endpoints
    - `POST /api/v1/external/tasks` — create and manage conversation sessions
    - `POST /api/v1/external/message` — send messages and receive agent responses
    - `wss://.../api/v1/external/session` — real-time streaming via WebSocket
    - Product recommendations and follow-up suggestions in responses
  </Step>
  <Step title="Mar 5 — CLI v0.1.77">
    **CLI** `beta`

    - Smart push with git-based diffing (PATCH, POST, PUT)
    - `aui dvalidate` for local schema validation
    - `aui pull-schema --improved` for coding agent schemas
    - `aui serve` web playground
    - `aui chat` interactive terminal conversations
  </Step>
</Steps>
